# Profitability and Pricing / 利润和定价

Use this reference when the user asks whether a SKU is worth selling, how to price, how tariffs affect margin, or how to build a launch budget.

当用户问 SKU 能不能做、怎么定价、关税如何影响利润或启动预算时使用本文件。

## Pricing Rule / 定价原则

Do not judge a SKU by factory cost alone. For TikTok Shop US, price decisions should include landed cost, platform fees, fulfillment, returns, creator commission, ad allowance, content production, and cash-flow timing.

不要只看出厂价判断 SKU。美区 TikTok Shop 定价要同时考虑到岸成本、平台费、履约、退货、达人佣金、广告预留、内容制作和现金流周期。

## Planning Formula / 规划公式

`gross profit = selling price - landed cost - platform/referral fee - fulfillment - payment/service fees - creator commission - ad allowance - return allowance - packaging/content allowance`

`gross margin = gross profit / selling price`

`break-even ROAS = selling price / gross profit before ads`

`毛利 = 售价 - 到岸成本 - 平台/佣金费 - 履约费 - 支付/服务费 - 达人佣金 - 广告预留 - 退货预留 - 包装/内容预留`

`毛利率 = 毛利 / 售价`

`保本 ROAS = 售价 / 扣广告前毛利`

## Decision Rules / 决策规则

- Reject the SKU if tariff classification is unclear and broker confirmation is missing.
- Reject or delay the SKU if product claims require medical, supplement, drug, baby/child safety, or regulated proof the seller cannot provide.
- Reject the SKU if expected refunds or shipping damage erase margin.
- Test with low inventory if the content angle is strong but margin or conversion is uncertain.
- Scale only after the listing, fulfillment, refund rate, creator content, and margin all survive the first data cycle.

- HTS/关税归类不清、没有报关行确认时，不要做大货。
- 医疗、保健品、药品、婴童安全或强监管宣称拿不出证明时，拒绝或延后。
- 退货和运输破损会吃掉利润时，不要硬上。
- 内容角度强但利润或转化不确定时，只做低库存测试。
- 只有链接、履约、退款率、达人内容和利润都通过第一轮数据，才放大。

## Launch Budget / 启动预算

| Bucket / 预算项 | What to Include / 包含内容 | Notes / 备注 |
|---|---|---|
| Product test / 测品 | Samples, first 5-20 SKUs / 样品和首批 SKU | Keep inventory small until data validates / 数据验证前低库存 |
| Compliance / 合规 | Broker, lab report, labels, IP check / 报关行、检测、标签、IP | Depends on category / 看类目 |
| Content / 内容 | Props, creator samples, editing / 道具、达人样品、剪辑 | Track cost per usable asset / 记录可用素材成本 |
| Fulfillment / 履约 | Warehouse, FBT/FBS, packaging / 仓库、FBT/FBS、包装 | Compare speed and cost / 比速度和成本 |
| Ads/creator / 广告达人 | Spark/GMV Max/affiliate commission / 广告和达人佣金 | Start with stop-loss rules / 先设止损 |
