# Official Sources / 官方来源

Use public, official sources for rule-sensitive claims. Prefer the current Seller Center and Seller University pages the user can access.

涉及平台规则、审核、物流、费用、退货、商品限制、广告和达人规则时，优先使用公开官方来源，并以用户当前能打开的 Seller Center 和 Seller University 页面为准。

## Verification Notes / 核对原则

- Platform rules change. For onboarding, logistics, fees, returns, product restrictions, ads, and creator rules, tell the user to verify the current Seller Center screen before execution.
- 平台规则会变化。入驻、物流、费用、退货、商品限制、广告和达人规则，都要以当前后台和官方页面为准。
- If the user asks for “latest” or asks about a rule that may have changed, check official sources before answering.
- 如果用户问“最新”或规则可能变化，先查官方来源再回答。
- If the official source conflicts with a course, screenshot, note, or older document, prefer the official source.
- 如果官方来源与课程、截图、笔记或旧文档冲突，以官方来源为准。
- For tariffs, never rely on a general product category name. Use the 10-digit HTSUS classification, country of origin, entry date, entry type, and current Chapter 99 headings. Ask the user to confirm with a customs broker for real shipments.
- 关税不要只按“大类”判断。必须看 10 位 HTSUS 编码、原产国、入境日期、报关方式和当前 Chapter 99 附加税号；实际出货前让用户找报关行确认。
