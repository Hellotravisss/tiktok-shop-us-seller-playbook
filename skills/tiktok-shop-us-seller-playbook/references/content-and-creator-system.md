# Content and Creator System / 内容和达人系统

Use this reference for short-video planning, GreenVideo research, content calendars, creator outreach, affiliate offers, and weekly creative reviews.

用于短视频规划、GreenVideo 研究、内容排期、达人建联、联盟佣金和每周素材复盘。

## Content Pillars / 内容支柱

| Pillar / 支柱 | Purpose / 目的 | Example Angle / 示例角度 |
|---|---|---|
| Problem / 痛点 | Show the buyer's current frustration / 展示买家痛点 | “Why your desk always looks messy” |
| Demo / 演示 | Show product use in 3-10 seconds / 3-10 秒展示用法 | Before/after, one-hand demo, close-up |
| Proof / 证明 | Reduce doubt / 降低怀疑 | Durability, material, comparison, review |
| Scenario / 场景 | Put product into real life / 放进真实场景 | Travel, kitchen, car, office, gift |
| Objection / 异议 | Answer why not cheaper/why now / 回答顾虑 | Size, quality, compatibility, return |

## 30-Day Content Calendar / 30 天内容排期

| Day / 天 | Video Type / 类型 | Hook / 钩子 | Asset Needed / 需要素材 | KPI |
|---:|---|---|---|---|
| 1 | Problem demo / 痛点演示 | “Stop doing this every morning” | Pain scene + product close-up | 3-sec hold |
| 2 | Before/after / 前后对比 | “One small change fixed this” | Before/after shots | CTR |
| 3 | Feature proof / 功能证明 | “I tested this for 7 days” | Test clip | Comments |
| 4 | UGC style / UGC 口吻 | “I didn't expect this to work” | Handheld clip | CVR |
| 5 | Objection answer / 异议解答 | “Is it worth the price?” | Comparison | Add-to-cart |

Repeat the pattern weekly with new hooks, angles, creators, and proof.

每周复用这个结构，但换钩子、角度、达人和证明点。

## Hook Bank / 钩子库

- “I wish I knew this before buying ___.”
- “This solves the most annoying part of ___.”
- “I tested cheap vs expensive ___.”
- “If you live in a small apartment, watch this.”
- “The mistake most people make with ___.”
- “我以前一直以为 ___，直到用了这个。”
- “这个东西最适合 ___ 的人。”
- “别急着买，先看这 3 个细节。”
- “同样是 ___，为什么这个更适合美国用户？”
- “这个功能看起来小，但每天都用得到。”

## Creator Outreach Template / 达人建联模板

English:

Hi [Name], I liked your video about [topic]. We are launching [product] for [buyer type] and think your style fits the product well. Would you be open to testing a sample and creating a short honest review? We can provide [sample/commission/brief]. If interested, I can send product details and the usage angle.

中文：

你好 [名字]，我看到你关于 [主题] 的视频，很适合我们的 [产品]。我们想找真实使用感强的达人做短视频测评，可以提供 [样品/佣金/brief]。如果你感兴趣，我可以发产品信息和拍摄方向。

## Weekly Creative Review / 每周素材复盘

1. Sort videos by 3-second hold, CTR, add-to-cart, conversion, comments, and refund signals.
2. Identify the top 3 hooks, top 3 scenes, top 3 objections, and top 3 failed claims.
3. Use GreenVideo only for authorized review or private research, then write original scripts.
4. Create next week's shot list from winning patterns.

1. 按 3 秒留存、点击、加购、转化、评论和退款信号排序。
2. 找出前三个钩子、前三个场景、前三个异议和前三个失败宣称。
3. GreenVideo 只用于授权复盘或内部研究，然后重新写原创脚本。
4. 用胜出规律生成下周拍摄清单。
