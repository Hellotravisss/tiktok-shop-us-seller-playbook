# Playbook Structure / 手册结构

Use this chapter order for a complete beginner-friendly English/Chinese playbook.

下面是适合完整新手教程的中英文双语章节结构。

## Part 1: Setup / 第一部分：开店准备

1. How to Use This Playbook and the Full Roadmap / 使用方法和完整路线图
2. TikTok Shop US Business Model, Roles, and Beginner Mistakes / 美国 TikTok Shop 模式、角色和新手误区
3. Business Entity, Tax, Bank, Warehouse, and Return Address Checklist / 主体、税务、银行、仓库、退货地址资料清单
4. Phone, Computer, Email, Account Permissions, and Work Environment / 手机、电脑、邮箱、账号权限和工作环境
5. Shop Registration, Review, and Basic Seller Center Setup / 店铺注册、审核、后台基础设置

## Part 2: Product Loop / 第二部分：商品闭环

6. Product Selection: Demand, Content Fit, Compliance, Profit, Supply Chain / 选品框架：需求、内容表现、合规、利润、供应链
7. Supply Chain, Samples, QC, Packaging, and Inventory / 供应链、样品、质检、包装和库存
8. Import Duties, China Tariffs, HTS Codes, and Landed Cost / 进口税费、中美关税、HTS 编码和到岸成本
9. Product Listing: Category, Title, Images, Selling Points, Price, Stock / 商品上架：类目、标题、图片、卖点、价格、库存
10. Logistics, Fulfillment, Returns, Customer Service, and Reviews / 物流、履约、退货、客服和评价

## Part 3: Growth Loop / 第三部分：增长闭环

11. TikTok Account Positioning, GreenVideo, and Content Asset System / TikTok 账号定位、GreenVideo 和内容素材系统
12. Short Video Scripts, Shooting, Editing, Publishing, and Review / 短视频脚本、拍摄、剪辑、发布和复盘
13. Creator/Affiliate Outreach, Samples, Commission, and Review / Creator/Affiliate 达人建联、寄样、佣金和复盘
14. Ads Basics: Budget, Creatives, Objectives, Data Decisions / 广告投放入门：预算、素材、目标、数据判断
15. Livestream Selling Prep: Product Mix, Host Flow, Scripts, Review / 直播带货准备：货盘、场控、话术、复盘

## Part 4: Operating System / 第四部分：运营系统

16. Daily and Weekly Operations SOP / 每日/每周运营 SOP
17. Data Dashboard: Traffic, Clicks, Add-to-Cart, Conversion, Profit / 数据看板：流量、点击、加购、转化、利润
18. Risk Control: Policy, Account Health, Violations, Stockouts, Bad Reviews / 风险控制：政策、健康分、违规、断货、差评
19. 30/60/90-Day Execution Plan / 30/60/90 天执行计划
20. Workbook and Templates / 实操工作簿和模板

## Chapter Formula / 单章公式

For each chapter, include:

- Objective / 本章目标
- Who should read this / 适合谁读
- What to prepare / 要准备什么
- Step-by-step workflow / 一步一步怎么做
- Verification checklist / 检查标准
- Common mistakes / 常见错误
- Done criteria / 本章完成标准
